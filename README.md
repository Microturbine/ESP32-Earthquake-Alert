# ESP32 EWS & QZSS Hybrid Decoder

ESP32のデュアルコアを活用し、地上（FM EWS）と宇宙（みちびき L1S）の両面から緊急事態を監視する、完全オフラインのデコーダーです。

## 主な特徴

- **ハイブリッド二重待機**: 
    - **FMラジオ EWS**: 64bps FSK信号をアナログ解析し、警報・地域・日時を抽出。
    - **みちびき L1S (災危通報)**: QZSS衛星からの生データ（サブフレーム）を捕捉・デコード。
- **QZSSデコード**:
    - **MT43 (DCR: 災危通報)**: 仕様書(IS-QZSS-DCR-016)に準拠。緊急地震速報（M・震度）や津波警報（高さ）のビット解析に加え、震央地名・津波予報区の日本語表示に対応。
    - **MT44 (DCX: 拡張災危通報)**: 仕様書(IS-QZSS-DCX-003)に準拠。Jアラート（国民保護情報）およびLアラート（自治体避難情報）の受信と解析に対応。
- **UIとノイズ対策**:
    - **ST7789P3 バー液晶対応**: LovyanGFXによる超スリムUI（284x76）を実装。
    - **サイレント描画（差分更新）**: 画面の情報が変化した時のみSPI通信を行うことで、アナログ音声へのノイズ混入を抑制。
    - **Wi-Fiレス設計**: 電波法（技適）への配慮とFMラジオへのノイズ混入を防ぐため、Wi-Fiは使用せずシリアル通信を用いたNVS（不揮発性メモリ）設定システムを採用。

## ハードウェア構成

### 1. 必要部品
- **ESP32 開発ボード**
- **RDA5807M** (FMラジオモジュール)
- **NEO-M8N** (GPSモジュール / FW 3.01以降推奨)
- **ST7789P3** (IPSカラー液晶モジュール / 76x284 バータイプ)
- **1.65Vバイアス回路** (音声入力用)

### 2. ピン接続

| ESP32 ピン | デバイス | 機能 | 備考 |
| :--- | :--- | :--- | :--- |
| **D19 / D18** | RDA5807M | SDA / SCL | I2C (Mutex保護) |
| **D22 / D23** | NEO-M8N | TXD / RXD | GPSデータ送受信 |
| **D4 (GPIO4)** | 音声入力 | ADC | バイアス回路経由 |
| **D14 / D13** | ST7789P3 | SCL / SDA | SPI (SCK / MOSI) |
| **D27 / D15 / D5** | ST7789P3 | RST / DC / CS | ※D17等の競合ピンを回避 |
| **D21** | ST7789P3 | BL | バックライト |
| **D2 (GPIO2)** | 本体LED | アラート表示 | 警報受信時に点灯 |

## ソフトウェア仕様

### セットアップと操作
- 起動時にGPSモジュールへ `UBX-CFG-MSG` を送信し、災危通報 (RXM-SFRBX) を自動有効化します。
- PCのシリアルモニター（115200bps）から各種設定を行い、設定は**フラッシュメモリに永続化**されます。
  - **`help`** または **`?`** : 利用可能なコマンドと使い方のヘルプを表示
  - **`80.0`** : 選局 (MHz) を設定して保存
  - **`v10`** : 音量調整 (0-15) を設定して保存
  - **`area 13`** : ターゲット地域コード（例: 13=東京）を設定して保存
  - **`status`** : 現在の受信状態（周波数、音量、RSSI、設定地域）を表示

## 今後の課題
- `area` コマンドで設定した自地域コードに基づく、QZSSおよびEWSの「地域フィルタリング（自地域のみアラートを鳴らす機能）」の実装。
- ブザー等を用いた音響アラートの発報。
- I2S ADCを用いたFM音声サンプリングの安定化（ジッタ低減）。
