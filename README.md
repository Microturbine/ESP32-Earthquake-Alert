# ESP32 Emergency Warning System (EWS) Decoder

ESP32とFMラジオモジュール（RDA5807M）を使用して、日本国内のアナログ緊急警報放送（EWS）をデコード・解析するプロジェクトです。

## 主な機能

- **アナログEWSデコード**: FM放送のFSK信号（640Hz/1024Hz）をリアルタイム解析し、警報種別（津波・地震）・地域・日時を表示。
- **ラジオ操作機能**: シリアルモニターから周波数変更や音量調整ができ、日常的にFMラジオとして使用可能。
- **高精度解析**: Goertzelアルゴリズムにより、低負荷かつ高精度な周波数検出を実現。
- **堅牢な設計**: 電源投入時のタイミング問題を解決する2秒の待機時間とI2Cリトライ機能を搭載。

## ハードウェア構成

### 1. 構成部品
- **ESP32 開発ボード**
- **RDA5807M** FMラジオモジュール
- **1.65Vバイアス回路**: ラジオ出力をESP32のADC（GPIO 4）に入力するためのオフセット回路（10kΩ抵抗×2、10μFコンデンサ×1）。

### 2. ピン接続

| ESP32 ピン | RDA5807M ピン | 機能 |
| :--- | :--- | :--- |
| **3.3V** | VCC | 電源 (3.3V) |
| **GND** | GND | グランド |
| **D19 (GPIO 19)** | SDA | I2C データ |
| **D18 (GPIO 18)** | SCL | I2C クロック |
| **D4 (GPIO 4)** | L-OUT/R-OUT | バイアス回路経由で接続 |

※ **重要**: GPIO 21/22 ではなく、D19/D18 を使用する設定になっています。

## 使い方

1. `FM_STATION` を地域のFM局（NHK-FMなど）の周波数に合わせて書き込みます。
2. ライブラリマネージャーから `RDA5807` ライブラリ（Radio by Matthias Hertel 推奨）をインストールします。
3. シリアルモニター（115200bps）を起動し、以下のコマンドで操作します。
   - **`80.0`** : 周波数を80.0MHzに変更
   - **`v5`** : 音量を5に変更（0〜15）
   - **`status`** : 現在の周波数・音量・RSSI（受信強度）を表示

## 解析ロジック
- **同期検出**: 前置符号(1100/0011)と固定符号(0x0E6D/0xF192)の20bitパターンを監視。
- **地域・日時**: LSBファーストで送られるデータを仕様に基づき反転・テーブル照合して復元。
- **サンプリング**: 8000Hzサンプリング、1ビットあたり125サンプルの窓で演算。

## 注意事項
- 本装置は実験・学習用であり、実際の防災用としては市販の検定合格品を使用してください。
- 受信感度が低い場合は、アンテナ（約75cmの電線）を `ANT` ピンに接続し、位置を調整してください。
