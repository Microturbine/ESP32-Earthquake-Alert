# ESP32 EWS Decoder via FM Radio

ESP32とFMラジオモジュール（RDA5807M）を使用して、FM放送に含まれる緊急警報放送（EWS: Emergency Warning System）をデコードし、地震などの緊急事態を検知するプロジェクトです。

## 概要

このプロジェクトは、FMラジオの音声信号から64bpsのFSK信号を抽出し、Goertzelアルゴリズムを用いて1024Hzと640Hzの周波数を解析することで、EWS信号をデジタルデータとして復調します。

## 主な機能

- **64bps FSK復調**: サンプリングレート8000Hzで音声を解析し、バイナリデータを抽出。
- **同期ワード検出**: EWSの同期フラグ（`0xB135`）を検出し、メッセージの開始を特定。
- **メッセージ解析**: 分類コード（開始信号、テスト信号、終了信号）を判定し、シリアルモニタに出力。
- **リアルタイムモニタリング**: 受信中のビットストリームをリアルタイムにシリアル出力。

## 構成部品 (Hardware)

- **ESP32 DevKitC** 等
- **RDA5807M** (FMラジオモジュール)
- **オーディオ入力回路**: ESP32の `GPIO34` (ADC) にラジオの音声出力を接続するための回路（カップリングコンデンサやバイアス回路など）

## ピンアサイン (Pin Mapping)

| 機能 | ESP32 ピン | RDA5807M ピン |
| :--- | :--- | :--- |
| I2C SDA | GPIO 21 | SDA |
| I2C SCL | GPIO 22 | SCL |
| Audio Input | GPIO 34 (ADC1_CH6) | L-OUT/R-OUT |
| Power | 3.3V | VCC |
| Ground | GND | GND |

## ソフトウェア要件 (Dependencies)

以下のライブラリをArduino IDEのライブラリマネージャーからインストールしてください。

- **RDA5807**: FMラジオモジュールの制御用

## セットアップ

1. **ハードウェアの接続**: 上記ピンアサインに従って回路を組み立てます。
2. **周波数の設定**: `ESP32.ino` 内の `FM_STATION` を、お住まいの地域のNHK-FMなどの周波数に合わせて変更してください。
    ```cpp
    #define FM_STATION 8250 // 82.5MHz の場合
    ```
3. **書き込み**: Arduino IDEで `ESP32.ino` を開き、ESP32にアップロードします。
4. **確認**: シリアルモニタ（115200bps）を開き、同期ワードの検出やデコード結果を確認します。

## 注意事項

- 本プロジェクトは実験的なものであり、生命の安全を保証するものではありません。
- 実際の緊急時には、テレビ・ラジオの本放送や公共の警報システムに従ってください。
- EWS信号の受信には、良好な電波状況と適切な回路設計（ノイズ対策）が必要です。
