# ESP32 Emergency Warning System (EWS) Decoder

ESP32とFMラジオモジュール（RDA5807M）を使用して、日本国内のFM放送で運用されている緊急警報放送（EWS）をデコード・解析するプロジェクトです。

## 概要

このシステムは、FM放送の音声信号（モノラル/ステレオ）をESP32のアナログ入力（ADC）から取り込み、信号処理によってデジタルデータを復元します。ARIB B14規格に基づく64bps FSK信号をデコードし、警報の種類、対象地域、発生日時などを解析します。

## 主な機能

- **FSK復調 (Frequency Shift Keying)**: 
    - 64bpsのデータ伝送をリアルタイムにデコード。
    - **Logic 1**: 1024Hz / **Logic 0**: 640Hz を監視。
- **Goertzelアルゴリズム**: 高速フーリエ変換(FFT)より低負荷なアルゴリズムで特定周波数の強度を抽出。
- **EWSプロトコル解析 (ARIB B14)**: 
    - **同期検出**: 前置符号（1100/0011）と固定符号（Type I: 0x0E6D / Type II: 0xF192）を検出。
    - **警報種別判定**: 第一種開始信号（地震・要請）、津波警報（開始）、および終了信号の判別。
    - **詳細データ解析**:
        - **地域符号**: 全国47都道府県および地域共通符号の特定。
        - **日時符号**: 月・日・時および年（西暦下1桁）のデコード。
- **FMラジオ制御**: I2C経由でRDA5807Mの周波数や音量を動的に設定可能。

## ハードウェア構成

### 1. 構成部品
- **ESP32 開発ボード** (例: ESP32-DevKitC)
- **RDA5807M** FMラジオモジュール
- **オーディオ入力回路**: ラジオ出力をESP32のADC（GPIO 4）に入力するためのバイアス回路（1.65Vオフセット）。

### 2. ピン接続

| ESP32 ピン | RDA5807M ピン | 機能 |
| :--- | :--- | :--- |
| 3.3V | VCC | 電源 (3.3V) |
| GND | GND | グランド |
| GPIO 21 | SDA | I2C データ |
| GPIO 22 | SCL | I2C クロック |
| GPIO 4 | L-OUT (バイアス経由) | アナログ音声入力 |

## ソフトウェア解説

### デコード・ステートマシン
コード内では以下の2つの状態を遷移しながらデコードを行います：
1. **SEARCH_SYNC**: 20ビットの同期パターン（前置4bit + 固定16bit）を検索。
2. **DECODE_FRAME**: 同期後の96ビット（データ+誤り訂正用ビット）を取得し、ビット反転やテーブル照合を行って解析。

### 解析ロジック
- **地域判定**: 12ビットの地域符号をビット反転（LSB first）して、定義済みの都道府県テーブルと照合。
- **日時判定**: 5ビット単位で構成される特殊な符号テーブルを用い、月・日・時・年を復元。
- **サンプリング**: 8000Hzでサンプリングし、1ビットあたり125サンプルの窓で周波数解析。

## 使い方

1. お住まいの地域のNHK-FMなどの周波数に合わせて `FM_STATION` を書き換えます（例: 82.5MHzなら `8250`）。
2. ライブラリマネージャーから `RDA5807` ライブラリをインストールします。
3. コードをESP32に書き込み、シリアルモニタを起動します（ボーレート: **115200**）。
4. 信号を受信すると、シリアルモニタに「地域」「日付」「時刻」などの解析結果が表示されます。

## 注意事項

- 本装置は実験・学習用であり、実際の防災用としては市販の型式検定合格品を使用してください。
- 受信環境（ノイズや電波強度、バイアス回路の精度）によってデコード精度は大きく左右されます。
