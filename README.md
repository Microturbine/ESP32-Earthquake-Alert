# ESP32 EWS & QZSS Hybrid Decoder

ESP32のデュアルコアを活用し、地上波（FM EWS）と宇宙波（みちびき L1S）の両面から緊急事態を監視する、高信頼性・完全オフラインの防災デコーダーです。

## 主な特徴

- **ハイブリッド二重待機**: 
    - **FMラジオ EWS**: 64bps FSK信号をアナログ解析し、警報・地域・日時を抽出。
    - **みちびき L1S (災危通報)**: QZSS衛星からの生データ（サブフレーム）を捕捉・デコード。
- **デュアルコア・パラレル処理**:
    - **Core 1 (App)**: 厳密なタイミングが要求される音声解析・FSK復調に専念。
    - **Core 0 (Pro)**: GPS受信、UBXバイナリパース、シリアルコマンド処理、および液晶描画を担当。
- **UIとノイズ対策**:
    - **ST7789P3 バー液晶対応**: LovyanGFXによる超スリムUI（284x76）を実装。
    - **ダブルバッファリング**: スプライト描画により、画面のチラつき（フリッカー）を排除。
    - **サイレント描画（差分更新）**: 画面の情報が変化した時のみSPI通信を行うことで、アナログ音声へのノイズ混入を抑制。
- **他機能**:
    - **I2C Mutex**: 排他制御により、コア間でのラジオモジュール競合とクラッシュを防止。
    - **アラート連携**: 信号検知時に液晶表示の切り替えと本体LED（GPIO2）による視覚的アラートを発報。

## ハードウェア構成

### 1. 必要部品
- **ESP32 開発ボード**
- **RDA5807M** (FMラジオモジュール)
- **NEO-M8N** (GPSモジュール / FW 3.01以降推奨)
- **ST7789P3** (IPSカラー液晶モジュール / 76x284 バータイプ)
- **1.65Vバイアス回路** (音声入力用)

### 2. ピン接続

| ESP32 ピン | デバイス | 機能 | 備考 |
| :--- | :--- | :--- | :--- |
| **D19 / D18** | RDA5807M | SDA / SCL | I2C (Mutex保護) |
| **D22 / D23** | NEO-M8N | TXD / RXD | GPSデータ送受信 |
| **D4 (GPIO4)** | 音声入力 | ADC | バイアス回路経由 |
| **D14 / D13** | ST7789P3 | SCL / SDA | SPI (SCK / MOSI) |
| **D27 / D15 / D5** | ST7789P3 | RST / DC / CS | ※D17等の競合ピンを回避 |
| **D21** | ST7789P3 | BL | バックライト |
| **D2 (GPIO2)** | 本体LED | アラート表示 | 警報受信時に点灯 |

## ソフトウェア仕様

### セットアップと操作
- 起動時にGPSモジュールへ `UBX-CFG-MSG` を送信し、災危通報 (RXM-SFRBX) を自動有効化します。
- シリアルモニター（115200bps）から以下のコマンドで操作可能です。
  - **`80.0`** : 選局 (MHz)
  - **`v10`** : 音量調整 (0-15)
  - **`status`** : 現在の受信状態（RSSI、周波数等）を表示
