# ESP32 EWS & QZSS Hybrid Decoder

ESP32のデュアルコアを活用し、地上波（FM EWS）と宇宙波（みちびき L1S）の両面から緊急事態を監視する、高信頼性・完全オフラインの防災デコーダーです。

## 主な特徴

- **ハイブリッド二重待機**: 
    - **FMラジオ EWS**: 64bps FSK信号をアナログ解析し、警報・地域・日時を抽出。
    - **みちびき L1S (災危通報)**: QZSS衛星からの生データ（サブフレーム）を直接デコード。
- **デュアルコア・パラレル処理**:
    - **Core 1 (App)**: 厳密なタイミングが要求される音声解析・FSK復調に専念。
    - **Core 0 (Pro)**: GPS受信、UBXバイナリパース、シリアルコマンド処理を担当。
- **エンタープライズ級の堅牢性**:
    - **I2C Mutex**: 排他制御により、コア間でのラジオモジュール競合とクラッシュを防止。
    - **Zero fragmentation**: 文字列処理に `String` クラスを使用せず、静的バッファを用いることで年単位の連続稼働が可能。

## ハードウェア構成

### 1. 必要部品
- **ESP32 開発ボード**
- **RDA5807M** (FMラジオモジュール)
- **NEO-M8N** (GPSモジュール / FW 3.01以降推奨)
- **1.65Vバイアス回路** (音声入力用)

### 2. ピン接続

| ESP32 ピン | デバイス | 機能 | 備考 |
| :--- | :--- | :--- | :--- |
| **D19 / D18** | RDA5807M | SDA / SCL | I2C (Mutex保護) |
| **D22 (RX)** | NEO-M8N | TXD | GPSデータ受信 |
| **D23 (TX)** | NEO-M8N | RXD | UBXコマンド送信 |
| **D4 (GPIO4)** | 音声入力 | ADC | バイアス回路経由 |

## ソフトウェア仕様

### セットアップ
起動時にGPSモジュールへ `UBX-CFG-MSG` コマンドを送信し、災危通報の解読に必要な **RXM-SFRBX (Raw Subframe)** メッセージを自動的に有効化します。

### シリアル操作
シリアルモニター（115200bps）から以下のコマンドで操作可能です。
- **`80.0`** : 選局 (MHz)
- **`v10`** : 音量調整 (0-15)
- **`status`** : 現在の受信状態（RSSI、周波数等）を表示
