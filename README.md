# ESP32 Emergency Earthquake Early Warning (EEW) Chime Detector

ESP32とFMラジオモジュール（RDA5807M）を使用して、NHKの緊急地震速報（EEW）で流れる特有のチャイム音をリアルタイムに検知・通知するプロジェクトです。

## 概要

このシステムは、FM放送の音声信号をESP32のアナログ入力（ADC）から取り込み、デジタル信号処理（DSP）によってチャイム音の和音構成を解析します。NHKが放送する緊急地震速報の「ピ・ポ・ピ・ポ」という音を正確に識別し、誤検知を最小限に抑えつつ発報します。

## 主な機能

- **和音解析 (Goertzelアルゴリズム)**: 
    - チャイムを構成する2つの和音（Cメジャー / C#メジャー）の各周波数成分を抽出。
    - **和音1 (C6/E6/G6)**: 1046.5Hz, 1318.5Hz, 1568.0Hz
    - **和音2 (C#6/F6/A6)**: 1108.7Hz, 1396.9Hz, 1760.0Hz
- **高精度スペクトル解析**:
    - **N=500 (16Hz分解能)**: サンプリングデータ数を増やすことで、近接する周波数を明確に分離。
    - **ハン窓 (Hanning Window)**: 窓関数の適用により、周波数の漏れ（リーク）を抑制し判定精度を向上。
- **誤検知防止ロジック**:
    - **2回連続検知条件**: 「ピ・ポ」のセットが2回連続で確認された場合のみ最終的な警報を出します。
    - **パワー比較判定**: 単純なしきい値だけでなく、和音同士の強度比を比較することでノイズによる誤作動を防止。
- **視覚的通知**:
    - **内蔵LED (GPIO 2)**: 検知時に点灯。
    - **自動消灯**: 発報から5秒後に自動で消灯し、待機状態に復帰。

## ハードウェア構成

### 1. 構成部品
- **ESP32 開発ボード** (例: ESP32-DevKitC)
- **RDA5807M** FMラジオモジュール
- **オーディオ入力回路**: ラジオ出力をESP32のADC（GPIO 4）に入力するためのバイアス回路（1.65Vオフセット推奨）。

### 2. ピン接続

| ESP32 ピン | RDA5807M ピン | 機能 |
| :--- | :--- | :--- |
| 3.3V | VCC | 電源 (3.3V) |
| GND | GND | グランド |
| GPIO 21 | SDA | I2C データ |
| GPIO 22 | SCL | I2C クロック |
| GPIO 4 | L-OUT | アナログ音声入力 |
| GPIO 2 | (内蔵LED) | 検知通知用LED |

## ソフトウェア解説

### 検知ステートマシン
1. **CHIME_IDLE**: 和音1（ピ）の開始を待機。
2. **CHIME_DETECTING_C1**: 和音1を検知中。一定時間継続すれば「和音2（ポ）」の待ち受けに移行。
3. **判定完了**: 和音2まで検知されると1セット（Pair 1）完了。これが2回繰り返されると最終発報。

### 解析パラメータ
- **サンプリングレート**: 8000Hz
- **解析サイズ**: 500サンプル (62.5ms)
- **判定比率**: 有効な和音が他方の和音の1.5倍以上の強度であること。

## 使い方

1. お住まいの地域のNHK-FMなどの周波数に合わせて `FM_STATION` を書き換えます。
2. コードをESP32に書き込み、シリアルモニタ（115200bps）を起動します。
3. チャイム音が流れると、シリアルモニタにデバッグ情報が表示され、検知完了時にLEDが点灯します。

## 注意事項

- 本装置は実験・学習用であり、実際の防災用としては気象庁認定の専用端末やスマートフォンアプリを併用してください。
- 受信環境や音声レベルによって検知感度が変わるため、必要に応じて `CHIME_THRESHOLD` を調整してください。
